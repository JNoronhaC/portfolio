      Projeto

      O projeto TaskEasy é um aplicativo criado para ajudar pessoas a organizarem suas tarefas do dia a dia de forma simples e prática. Muitas pessoas têm dificuldades em lembrar compromissos, trabalhos ou atividades importantes, e por isso a proposta do aplicativo é oferecer uma ferramenta digital que facilite o planejamento das tarefas diárias. A ideia principal do projeto é permitir que o usuário registre suas atividades, acompanhe o que precisa ser feito e visualize o que já foi concluído. Dessa forma, o aplicativo ajuda a melhorar a organização pessoal e aumentar a produtividade. O aplicativo permite criar tarefas, editar informações, adicionar prazos e marcar atividades como concluídas. Também é possível organizar as tarefas por categorias, facilitando a visualização de compromissos relacionados a estudo, trabalho ou atividades pessoais. O público-alvo do projeto são estudantes, profissionais e qualquer pessoa que precise organizar melhor sua rotina. O aplicativo foi pensado para ser simples de usar, com uma interface clara e acessível para qualquer usuário. Para o desenvolvimento do projeto podem ser utilizadas tecnologias como HTML, CSS e JavaScript para a parte visual do aplicativo, além de Node.js para o funcionamento do sistema e um banco de dados como SQLite ou MongoDB para armazenar as tarefas. Com este projeto espera-se oferecer uma solução prática que ajude as pessoas a planejarem melhor seu tempo, evitando esquecimentos e tornando a rotina mais organizada. O TaskEasy demonstra como a tecnologia pode ser utilizada para melhorar a gestão das atividades do dia a dia.
